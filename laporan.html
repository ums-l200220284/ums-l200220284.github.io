  2. Membuat repository baru dengan nama 'ums-l200220284.github.io'.
  3. Menginstall Github for Windows pada komputer kerja, https://git-scm.com/downloads/win.
  4. Pada windows jalankan perintah git-CMD.
  5. Pada terminal git, jalankan perintah git clone [alamat repository 'ums-l200220284.github.io'].

  7. Menjalankan perintah wsl -l -o untuk mengetahui distro Linux yang tersedia pada wsl.
